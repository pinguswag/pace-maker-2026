# 온보딩 가이드 구현

## 구현 완료 사항

### 1. Guide 페이지 (`app/guide/page.tsx`)
- 5단계 온보딩 체크리스트 구현
- 각 단계에 설명과 CTA 버튼 포함
- 프로젝트/작업 개수 확인하여 완료 상태 표시
- 인증 게이팅 적용 (로그인 필요)
- 한국어 UI

### 2. Home 페이지 Empty State (`app/page.tsx`)
- 프로젝트가 0개이거나 작업이 0개일 때 온보딩 패널 표시
- "전체 가이드 보기" 링크 추가
- 프로젝트/작업 개수 실시간 확인
- 조건부 메시지 및 CTA 버튼

## 5단계 온보딩 체크리스트

1. **프로젝트 생성**
   - 프로젝트가 1개 이상이면 완료 표시
   - CTA: "프로젝트 만들기" → `/projects/new`

2. **작업 3개 생성**
   - 작업이 3개 이상이면 완료 표시
   - 프로젝트가 없으면 "먼저 프로젝트 만들기" 표시
   - CTA: "작업 만들기" → `/projects/{id}` 또는 `/projects/new`

3. **이번 주에 추가**
   - 주간 계획에 작업 추가 안내
   - CTA: "주간 계획 보기" → `/`

4. **오늘로 선택**
   - 오늘 탭에서 작업 선택 안내
   - CTA: "오늘 탭 보기" → `/`

5. **작업 완료 및 리뷰**
   - 작업 완료 및 리뷰 확인 안내
   - CTA: "리뷰 보기" → `/review`

## 파일 변경 사항

### 새로 생성된 파일
- `app/guide/page.tsx` - 온보딩 가이드 페이지

### 수정된 파일
- `app/page.tsx` - Empty State 가이드 추가

## 스타일링

- Tailwind CSS 사용
- 그라데이션 배경 (`from-blue-50 to-indigo-50`)
- 완료된 단계는 초록색 강조
- 반응형 디자인

## 인증 게이팅

- Guide 페이지는 로그인 필요
- 미인증 사용자는 `/login`으로 리다이렉트
- Home 페이지의 Empty State도 인증 후에만 표시
